# 📊 Análise de Dados - COVID-19

## 🔍 Descrição
Este projeto consiste na análise exploratória e geração de insights sobre o COVID-19, utilizando ferramentas como Python, SQL e Power BI.

## 🎯 Objetivo
Identificar padrões, realizar análise de tendências, gerar dashboards interativos.

## 🔗 Fonte de Dados
https://data.brasil.io/dataset/covid19/caso_full.csv.gz

## 🧠 Tecnologias e Ferramentas
- Python (pandas, numpy, matplotlib, seaborn)
- SQL (consultas, filtros, joins)
- Power BI
- Jupyter Notebook
- Git e GitHub

## 🚀 Etapas Realizadas
1. Coleta dos Dados
2. Limpeza e Tratamento
3. Análise Exploratória
4. Visualizações
5. Desenvolvimento de Dashboard Interativo
6. Conclusões
